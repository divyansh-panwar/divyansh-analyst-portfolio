## Overview
This project analyzes e-commerce sales data using Python, Pandas and power bi to generate meaningful business insights.

## Objectives
- Calculate total revenue
- Analyze category-wise sales
- Analyze city-wise sales
- Identify top-selling products
- Identify top revenue-generating products

## Analysis Performed
1. Data Loading using Pandas
2. Revenue Calculation
3. Category-wise Revenue Analysis
4. City-wise Revenue Analysis
5. Top Selling Product Analysis
6. Data Visualization using Matplotlib

## Key Insights
- Electronics generated the highest revenue.
- Major sales came from Delhi and Mumbai.
- Mobile and Laptop were among the highest revenue-generating products.
